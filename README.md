# Proyecto Inteligencia Artificial
## Conducción Autónoma

### Ejecución:
Para ejecutar solamente el environment se tiene que definir 
```
play = True en main()
```

### Entrenar:
Para entrenarse define 
```
load = False
```
Para entrenar usando como base una red ya entrenada definir 
```
load = True
```

### Ventana:
El tamaño de la ventana esta harcoded por la necesidad del agente de recibir un tamaño de imagen fijo
